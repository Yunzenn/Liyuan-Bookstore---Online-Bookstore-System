## 3. 第二层数据流图

### 3.1 客户管理子系统

```
    ┌────────┐                              ┌────────┐
    │        │  ──注册请求──▶  ┌──────┐     │        │
    │        │                 │ 验证 │     │        │
    │        │  ◀──验证结果──  │ 信息 │     │        │
    │        │                 └──┬───┘     │        │
    │  客户  │                    │         │ 客户表 │
    │        │  ──登录请求──▶  ┌──▼───┐     │        │
    │        │                 │ 身份 │ ──▶ │        │
    │        │  ◀──登录结果──  │ 认证 │     │        │
    │        │                 └──────┘     └────────┘
    └────────┘
```

**处理说明：**
- **验证信息**：检查注册信息完整性、邮箱唯一性
- **身份认证**：验证用户名和密码，创建会话

### 3.2 图书管理子系统

```
    ┌────────┐                 ┌────────┐
    │        │ ──搜索请求──▶   │        │
    │        │                 │ 搜索   │ ──┐
    │        │ ◀──搜索结果──   │ 图书   │   │
    │  客户  │                 └────────┘   │    ┌────────┐
    │        │                              ├──▶ │        │
    │        │ ──浏览请求──▶  ┌────────┐   │    │ 图书表 │
    │        │                │ 获取   │ ──┘    │        │
    │        │ ◀──图书列表──  │ 图书   │        └────────┘
    │        │                │ 列表   │
    └────────┘                └────────┘
```

**处理说明：**
- **搜索图书**：按书名、作者、ISBN进行模糊搜索
- **获取图书列表**：分页返回图书目录

### 3.3 订单管理子系统

```
    ┌────────┐                              ┌────────┐
    │        │ ──下单请求──▶  ┌──────┐      │        │
    │        │                │ 创建 │ ──▶  │ 订单表 │
    │        │ ◀──订单确认──  │ 订单 │      │        │
    │        │                └──┬───┘      ├────────┤
    │  客户  │                   │          │ 订单   │
    │        │                   └──────▶   │ 明细表 │
    │        │                              └────────┘
    │        │ ──查询请求──▶  ┌──────┐
    │        │                │ 查询 │ ──▶ 订单表 + 订单明细表
    │        │ ◀──订单详情──  │ 订单 │
    │        │                └──────┘
    └────────┘
```

**处理说明：**
- **创建订单**：验证库存、计算金额、生成订单记录
- **查询订单**：按订单号或时间范围查询

## 4. 数据流描述

### 4.1 主要数据流

| 数据流 | 来源 | 去向 | 组成 |
|--------|------|------|------|
| 注册请求 | 客户 | 验证信息 | 客户名+地址+电话+邮箱+密码 |
| 登录请求 | 客户 | 身份认证 | 邮箱+密码 |
| 图书查询请求 | 客户 | 搜索图书 | 关键词+查询类型 |
| 下单请求 | 客户 | 创建订单 | 客户ID+[{ISBN, 数量}] |
| 订单查询请求 | 客户 | 查询订单 | 订单号/时间范围 |
| 图书信息 | 图书表 | 搜索图书 | ISBN+书名+作者+定价+库存 |
| 订单信息 | 订单表 | 查询订单 | 订单号+客户ID+时间+状态+金额 |

### 4.2 数据存储说明

| 数据存储 | 说明 | 主要字段 |
|----------|------|----------|
| 客户表 | 存储注册客户信息 | customer_id, customer_name, email, password_hash |
| 图书表 | 存储图书目录信息 | ISBN, title, author, price, stock |
| 订单表 | 存储订单头信息 | order_id, customer_id, order_date, status, total_amount |
| 订单明细表 | 存储订单项详情 | order_id, ISBN, quantity, unit_price |
