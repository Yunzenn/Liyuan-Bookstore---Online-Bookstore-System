# 网上书店数据库系统 - 开发文档

## 1. 项目概述

### 1.1 项目背景
实体书店"梨园书屋"需要从手工管理转型为在线管理系统，通过Web站点实现订单、出货、货款交付状态的在线管理。

### 1.2 项目目标
- 建立完整的网上书店数据库系统
- 实现客户注册、图书浏览、订单管理等核心功能
- 完成数据库设计全流程（需求分析→概念设计→逻辑设计→物理设计→实施测试）

## 4. 数据库设计概要

### 4.1 核心实体
| 实体 | 说明 | 主键 |
|------|------|------|
| 客户 (Customer) | 注册用户信息 | customer_id |
| 图书 (Book) | 图书目录信息 | ISBN |
| 订单 (Order) | 订单头信息 | order_id |
| 订单明细 (OrderItem) | 订单中具体图书项 | (order_id, ISBN) |

### 4.2 实体关系
- 客户 → 订单：**1:N**（一个客户可下多个订单）
- 订单 → 图书：**M:N**（通过订单明细表实现）
- 订单 → 订单明细：**1:N**（一个订单包含多个明细项）

### 4.3 关键表结构预览

**客户表 (customer)**
```sql
customer_id     INT PRIMARY KEY AUTO_INCREMENT  -- 客户唯一标识
customer_name   VARCHAR(50) NOT NULL            -- 客户姓名
address         VARCHAR(200)                    -- 地址
phone           VARCHAR(20)                     -- 电话
email           VARCHAR(100) UNIQUE             -- 邮箱
password_hash   VARCHAR(255) NOT NULL           -- 密码哈希
created_at      DATETIME DEFAULT CURRENT_TIMESTAMP
```

**图书表 (book)**
```sql
ISBN            VARCHAR(13) PRIMARY KEY         -- ISBN号
title           VARCHAR(200) NOT NULL           -- 书名
author          VARCHAR(100) NOT NULL           -- 作者
price           DECIMAL(10,2) NOT NULL          -- 定价
publisher       VARCHAR(100)                    -- 出版社
publish_year    YEAR                            -- 出版年
stock           INT DEFAULT 0                   -- 库存数量
```

**订单表 (order)**
```sql
order_id        INT PRIMARY KEY AUTO_INCREMENT  -- 订单号
customer_id     INT NOT NULL                    -- 客户ID
order_date      DATETIME DEFAULT CURRENT_TIMESTAMP -- 下单时间
status          ENUM('待付款','已付款','已发货','已完成','已取消') DEFAULT '待付款'
total_amount    DECIMAL(10,2)                   -- 订单总额
FOREIGN KEY (customer_id) REFERENCES customer(customer_id)
```

**订单明细表 (order_item)**
```sql
order_id        INT                             -- 订单号
ISBN            VARCHAR(13)                     -- ISBN号
quantity        INT NOT NULL                    -- 数量
unit_price      DECIMAL(10,2) NOT NULL          -- 单价（下单时价格）
PRIMARY KEY (order_id, ISBN),
FOREIGN KEY (order_id) REFERENCES order(order_id),
FOREIGN KEY (ISBN) REFERENCES book(ISBN)
```

## 5. 功能模块设计

### 5.1 客户管理模块
- **注册**：填写客户名、地址、电话、邮箱、密码
- **登录/登出**：基于Session的认证
- **个人信息查看/修改**

### 5.2 图书管理模块
- **图书列表**：分页展示、按书名/作者搜索
- **图书详情**：查看完整信息
- **库存管理**：管理员可增删改图书

### 5.3 订单管理模块
- **创建订单**：选择图书和数量，生成订单
  - 支持在一个订单中添加多种图书
  - 自动计算每项小计和订单总额
  - 下单时记录图书当前价格（unit_price）
- **查看订单**：客户查看自己的订单列表
- **订单详情**：查看订单明细和状态
- **订单状态更新**：管理员更新出货/付款状态

### 5.4 出货管理模块（需求要求）
- **出货状态跟踪**：待发货→已发货→已签收
- **发货操作**：管理员确认发货，记录发货时间
- **物流信息**：可选填物流单号

### 5.5 货款管理模块（需求要求）
- **付款状态跟踪**：待付款→已付款→已退款
- **付款确认**：管理员确认收到货款
- **付款方式**：支持记录付款方式（在线支付/货到付款）

### 5.6 订单状态流转图
```
┌─────────┐    付款    ┌─────────┐    发货    ┌─────────┐    确认    ┌─────────┐
│ 待付款   │ ────────▶ │ 已付款   │ ────────▶ │ 已发货   │ ────────▶ │ 已完成   │
└─────────┘           └─────────┘           └─────────┘           └─────────┘
     │                     │                     │
     │ 取消                │ 退款                │ 签收确认
     ▼                     ▼                     ▼
┌─────────┐           ┌─────────┐           ┌─────────┐
│ 已取消   │           │ 已退款   │           │ 已签收   │
└─────────┘           └─────────┘           └─────────┘
```

## 7. 环境配置

### 7.1 安装MySQL
```bash
# Windows: 下载MySQL Installer
# https://dev.mysql.com/downloads/installer/

# 创建数据库
CREATE DATABASE online_bookstore CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 7.2 安装Python依赖
```bash
pip install flask pymysql
```

### 7.3 配置数据库连接
在 `app/config.py` 中配置：
```python
MYSQL_HOST = 'localhost'
MYSQL_PORT = 3306
MYSQL_USER = 'root'
MYSQL_PASSWORD = 'your_password'
MYSQL_DB = 'online_bookstore'
```
